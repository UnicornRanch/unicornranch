### Para MACOS

Instale KUBECTL 
Instale e configure o AWSCLI 


#### Instalação EKSCTL 

Utilizando brew, instale o repositorio e depois instale o eksctl 
```
brew tap weaveworks/tap
brew install weaveworks/tap/eksctl
```

Verifique a instalação:
````
eksctl version
````
